# Microloan Portfolio Dashboard – Risk & Repayment Insights

### Author
**Haminatou Coulibaly**
Data Analyst | Ethical Finance & AI

---

## Project Overview

This project analyzes microloan portfolio performance to uncover insights about repayment behavior, client risk profiles, and portfolio health.

The dashboard was built in Power BI using real-world financial data to help lenders and microfinance institutions make data-driven, ethical lending decisions.

---

## Objectives
- Monitor loan disbursement trends and repayment performance
- Identify high-risk borrowers and repayment delays
- Analyze client behavior by demographics and loan terms
- Support ethical and inclusive finance decisions

---

## Dataset

| Field | Description |
|-------|-------------|
| customer_id | Unique identifier for each client |
| loan_amount | Total amount disbursed |
| loan_repaid | Amount repaid |
| loan_term_days | Duration of the loan |
| interest_rate | Interest rate applied |
| segment | Client risk segment (Low, Medium, High) |
| month_year | Disbursement date |

*Synthetic or anonymized data used for learning purposes.*

---

## Dashboard Pages
1. **Microloan Overview** - Portfolio summary and key performance metrics
2. **Client Insights & Risk Analysis** - Default rate, repayment ratio, and risk segmentation
3. **Loan Performance & Trends** - Monthly loan volume and repayment over time
4. **Client Segmentation & Behavior** - Analysis by gender, risk level, and repayment behavior

---

## Key Power BI Features Used
- DAX measures for KPIs (Repayment %, Average Loan Term, etc.)
- Slicers for interactive filtering (risk level, month, segment)
- KPI cards and line charts for performance tracking
- Custom color theme for visual consistency
- Page navigation buttons for an intuitive multi-page layout

---

## Insights Discovered
- Short-term loans have higher default probability
- Low-income clients tend to repay smaller loans faster
- Portfolio diversification across segments improves repayment rates

---
